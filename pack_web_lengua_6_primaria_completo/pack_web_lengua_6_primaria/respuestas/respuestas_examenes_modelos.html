MODELO 5 - PRÁCTICO (Respuestas)
PRONOMBRES Y DETERMINANTES
A) Determinante o Pronombre:
1. Mi → Det. (acompaña a "mochila")
2. tuya → Pron. / mía → Pron. (sustituyen a nombres)
3. Aquellos → Det. (acompaña a "libros")
4. ese → Pron. (no acompaña a ningún nombre)
5. Nuestros → Det. (acompaña a "amigos")
B) Clasificación de pronombres:
1. Nosotros → Personal
2. Alguien → Indefinido
3. Ese → Demostrativo
4. ¿Quién? → Interrogativo
5. Mío → Posesivo
POLISEMIA, HOMÓGRAFAS Y HOMÓFONAS
A) Significados de "banco":
1. Asiento para sentarse
2. Entidad financiera donde se guarda dinero
B) Homógrafas o homófonas:
1. Vino/vino → Homógrafas (se escriben igual)
2. Hola/ola → Homófonas (suenan igual, se escriben diferente)
3. Vaca/baca → Homófonas (suenan igual, se escriben diferente)
4. Cura/cura → Homógrafas (se escriben igual)
C) Truco homógrafas:
Sustantivo + Verbo

DIPTONGO E HIATO
1. Cielo → Diptongo (ie = cerrada+abierta)
2. País → Hiato (í con tilde = hiato seguro)
3. Cuidado → Diptongo (ui = cerrada+cerrada)
4. Río → Hiato (í con tilde = hiato seguro)
5. Fuerte → Diptongo (ue = cerrada+abierta)
6. Teatro → Hiato (ea = abierta+abierta)
Explicación "país":
Es hiato porque la vocal cerrada "í" lleva tilde, lo que rompe el diptongo

Desafío ortográfico - Texto corregido:
"Ayer fuimos a ver una película al cine. Mi hermano llevó su mochila con golosinas. Al salir, nos encontramos con nuestros amigos del colegio que también habían ido a ver la misma película."
Errores corregidos:
1. haver → a ver (preposición + verbo ver)
2. pelicula → película (llana terminada en vocal, no lleva tilde)
3. llevo → llevó (3ª persona, lleva tilde)
4. tambien → también (aguda terminada en n)
5. avian → habían (verbo haber con h)
Criterios de evaluación: Se valorará especialmente la precisión en el análisis morfosintáctico, la correcta identificación de figuras literarias y la aplicación de reglas ortográficas con su justificación correspondiente.
